SwellScore — Real-Time Surf Forecast System

Full-stack product — live in production

Full-stack web application delivering real-time surf forecasts for Maui. Built end-to-end: React + TypeScript frontend, Flask API backend, and a multi-source data pipeline integrating NOAA buoys, PacIOOS ERDDAP wave models, and weather APIs.

View Live Start a Project

The Challenge

Maui surfers had no reliable, break-specific forecast tool. Existing services used generic regional models that missed the nuance of individual breaks. Building one required integrating multiple data sources with different APIs, formats, and update cadences and making it fast enough for a mobile user checking conditions at the beach.

The Solution

Built a complete product from scratch: a React + TypeScript frontend optimized for mobile and a Flask API backend aggregating PacIOOS ERDDAP wave models, NOAA buoy readings, WRF wind forecasts, and tide data, with graceful fallbacks when primary sources go offline.

Results

  • Full-stack product live in production
  • Real-time forecasts for 10 Maui surf breaks
  • Multi-source data pipeline with graceful fallbacks
  • Mobile-optimized for on-beach use
